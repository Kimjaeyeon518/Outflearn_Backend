# 유신사 (Usinsa)
### `무신사 클론 SPA 프로젝트`

-------------------------------------

# Tech-Stack
## Backend (~ing)
* Java 11
* Spring Boot 2.6.3
* REST API
* MySQL + MongoDB
* Spring Security + Redis(예정) + JWT + OAuth2(네이버, 카카오, 구글)
* Spring Data JPA
* Swagger 3.0.0
* Thymeleaf

## Frontend (예정)
* React 5.0.0
* TypeScript

--------------------------------------
